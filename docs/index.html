GEODE Layers
System Overview
Hierarchical Multi-Agent Architecture

상위 Orchestrator Super-Graph가 목표를 분해하고 하위 Worker(SelfHealingGraph)를 병렬 통제. Quality Gate가 산출물을 검수하며, SessionLane이 자원 충돌을 방지합니다.

GEODE Reference Model
5-Layer Verification Pipeline

에이전트의 모든 출력은 5개의 결정론적 관문을 순차적으로 통과해야 합니다. 각 계층은 독립적 검증 책임을 가집니다.

L1 Atomic XML Matching LLM 출력이 <think>/<action>/<tool_call> 구조를 준수하는지 검증 parse_llm_output()
L2 Syntactic Broom AST 파싱으로 문법 오류를 샌드박스 전송 전 사전 차단 lint_code()
L3 Semantic Defense Jaccard 유사도 기반 의미론적 반복 감지 및 조기 차단 SemanticLoopDetector
L4 Self-Healing 실행 오류 시 stderr + 원본 목표로 자동 교정 루프 conditional_edges
L5 Runtime Smoke Test 격리 컨테이너에서 제한 시간 내 동작 물리적 증명 SandboxExecutor

Task 6 — Semantic Defense

Semantic Infinite Loop Detector

최근 N개의 <think> 내용과 도구 호출 인자의 Jaccard 유사도를 슬라이딩 윈도우로 계산. 동일한 헛발질이 임계치 이상 반복되면 SemanticLoopException을 발생시켜 max_attempts 도달 전에 조기 차단합니다.

defense/semantic.py 
avg_sim = self._average_pairwise_similarity()
if avg_sim >= self._threshold:
    raise SemanticLoopException(
        f"Semantic loop: {avg_sim:.2f} >= {self._threshold}"
    )

Task 8 — Infra Resilience

SessionLane & Jitter Backoff

다중 워커의 자원 경합을 막는 Per-key 세마포어(SyncSessionLane)와 Thundering Herd 현상을 방지하는 Decorrelated Jitter 지수적 백오프. ThreadPoolExecutor와 asyncio 환경 모두 지원합니다.

infra/session_lane.py 
with self._meta_lock:
    if key not in self._locks:
        self._locks[key] = threading.Lock()
lock = self._locks[key]
acquired = lock.acquire(timeout=self._timeout)
if not acquired:
    raise SessionLaneTimeout(f"timeout for {key}")

Module Architecture
Nine Pillars of Kappa

각 모듈은 독립 프로토콜 기반으로 설계되어 테스트 시 Mock으로 완전 교체 가능합니다.

$

kappa.budget

BudgetTracker — 스레드 안전 토큰/비용 추적. RLock 기반.
BudgetGate — LLM 호출 래퍼. pre_check → call → record_usage.
LLMProvider — Protocol. AnthropicProvider 구현체 제공.

S

kappa.sandbox

SandboxExecutor — 코드 문자열 → Docker 컨테이너 실행.
ContainerRuntime — Protocol. DockerRuntime 구현체.
SandboxResult — exit_code, stdout, stderr, timed_out.

G

kappa.graph

SelfHealingGraph V2 — 5-node LangGraph StateGraph.
AgentState — TypedDict + tool_calls, memory_context.
nodes.py — <action>/<tool_call> 분기 파서.

M

kappa.memory

VFSManager — 격리 작업공간 파일 시스템.
MemoryConfig — workspace_root, auto_inject_files.
Path Traversal — ../ 공격 원천 차단.

T

kappa.tools

ToolRegistry — 도구 등록/디스패치 + 예산 연동.
BaseTool — Protocol. 플러그 앤 플레이 인터페이스.
Builtins — read_memory, write_memory 시스템 도구.

D

kappa.defense

SemanticLoopDetector — Jaccard 유사도 감지기.
SemanticConfig — window_size, threshold, min_samples.
SemanticLoopException — 조기 차단 예외.

I

kappa.infra

SyncSessionLane — Per-key Lock 기반 자원 직렬화.
AsyncSessionLane — asyncio.Lock 비동기 버전.
jitter_backoff — AWS Decorrelated Jitter 백오프.

O

kappa.graph.orchestrator

OrchestratorGraph — LangGraph Super-Graph 오케스트레이터.
Planner/Reviewer — LLM 기반 DAG 분해 & Quality Gate.
SubTask — TypedDict DAG 노드, 병렬 디스패치.

R

kappa.telemetry

TelemetryManager — 옵저버 패턴 JSONL 궤적 기록기.
TrajectoryRecord — think/critique/score 데이터클래스.
summary() — 집계 통계 (avg_score, rejection_rate).
